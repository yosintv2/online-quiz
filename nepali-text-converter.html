What is Nepali Unicode?

Tips and Tricks for Nepali Unicode Typing

  • Combine English into your Nepali Unicode text: Place English text inside curly braces {} to keep it unchanged. Example: yo {mobile} mero ho = यो mobile मेरो हो
  • Phonetically similar Nepali letters: Use specific notations like ta = त, Ta = ट, tha = थ, Tha = ठ, da = द, Da = ड, dha = ध, Dha = ढ, na = न, Na = ण, sha = श, Sha = ष. Case doesn't matter for other letters.
  • Separate letters: Use a slash (/) to separate letters if combined incorrectly. Example: pratishatko = प्रतिशत्को, pratishat/ko = प्रतिशतको
  • Add halant: Use a backslash (\) to add a halant. Example: bas = बस, bas\ = बस्
  • Special characters:
    • ri^ = रि (as in प्र)
    • rr = र्‍ (as in गर्‍य)
    • rri = ऋ
    • rree = ॠ
    • yna = ञ
    • chha = छ
    • ksha = क्ष
    • gya = ज्ञ
    • * = अनुस्वर
    • ** = चन्द्रबिन्दु
    • om = ॐ
